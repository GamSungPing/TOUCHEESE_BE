### Commit Convention

| 커밋 구분     | 설명                    |
|-----------|-----------------------|
| Feat	     | 기능 추가 및 개선            |
| Fix	      | 버그 수정                 |
| Docs	     | 코드 외 문서 작업            |
| Test	     | 테스트 추가/수정             |
| Build	    | 빌드 프로세스 관련 수정(yml)    |
| Refactor	 | 코드 정리/리팩토링 (기능 수정X)   |
| Merge	    | 머지/충돌 정리 (상세 설명 필수 X) |
| Comment	  | 주석 수정 및 추가 (코드 수정X)   |

ex/ `Feat: 개요 - 상세 설명(옵션)`
